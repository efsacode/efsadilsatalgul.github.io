Özet

Dönem: 23 Ocak 2023 - 12 Şubat 2023 Çalışma Şekli: Ofisten

Staj sürecimde hem Linux sistem yönetimi hem de front-end web geliştirme alanlarında çalıştım. Bu süreçte sistem yönetimi, ağ ve güvenlik işlemleri üzerine teknik beceriler geliştirirken, aynı zamanda front-end geliştirme konusunda pratik deneyim edindim.

Öne Çıkan Sonuçlar

Linux işletim sistemi üzerinde:

  • Sistem yönetimi
  • Genel komut kullanımı
  • Ağ işlemleri
  • Geliştirme ve çalışma ortamı yönetimi üzerine çalıştım.

Web sitesi geliştirme alanında:

  • API entegrasyonu ve veri alışverişi
  • Responsive (mobil uyumlu) tasarım
  • Front-end geliştirme üzerine çalıştım.

Kullanılan Teknolojiler

  • Linux İşletim Sistemi
  • HTML-CSS
